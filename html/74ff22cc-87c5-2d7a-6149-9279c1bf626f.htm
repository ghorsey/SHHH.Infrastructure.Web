Usage Note
In Visual Basic and C#, you can call this method as an instance method on any object of type HtmlHelper. When you use instance method syntax to call this method, omit the first parameter. For more information, see Extension Methods (Visual Basic) or Extension Methods (C# Programming Guide).
Examples

            Call to @Html.Stylesheet(Url.Content("~/Content/Styles/site"), "-min")
            Will result in the following HTML <link type="text/css" rel="stylesheet" href="/Content/Styles/site-min.css"/>
            when the debugger is not attached.  Otherwise, if the debugger is attached, it will result in the following HTML:
              <link type="text/css" rel="stylesheet" href="/Content/Styles/site.css"/>
